Chrome Extension Installation Guide

Follow these quick steps to install the Playlist Info Exporter Chrome Extension and unlock Google AI Mode.

1

Download the Extension

Download the packed extension zip file using the link below:

Download Extension ZIP
2

Extract the ZIP file

Locate the downloaded spotify-playlist-exporter.zip file on your computer and extract it to a permanent folder (e.g. inside Documents or a custom location).

3

Open Chrome Extensions

Open Google Chrome, type chrome://extensions in the address bar, and hit Enter. (Alternatively, click the Chrome Menu (three dots) -> Extensions -> Manage Extensions).

4

Enable Developer Mode

In the top-right corner of the Extensions page, turn on the Developer mode switch.

5

Load the Unpacked Folder

Click the Load unpacked button in the top-left corner of the Extensions page. In the file explorer, select the folder where you extracted the extension files (the folder containing manifest.json).

6

Verify and Refresh

Once loaded, go back to the Playlist Info Exporter page, do a hard refresh (Ctrl + Shift + R), and connect Spotify. The "Google AI Mode" option will now be active!
